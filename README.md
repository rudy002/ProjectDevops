## Usage/Examples

Open terminal from VSCODE or other editor.

```npm install```

Wait the installation of all package and do : 

```npm start```

```http://localhost:3000/``` 

Or go to the website deploiement in Azure :

```https://projectdevops2023rudy.azurewebsites.net/```
